Frameworks

AngularJS was utilized to create a responsive map element that updates a table with values through the use of data binding.

Bootstrap 4 was the backbone of the layout. Bootstrap handled much of the styling, and provided a useful grid system to display elements eloquently.

jQuery 4 provided extra functionality of our JavaScript, allowing us to have consistent date pickers, and shorthand, yet powerful, selections.

API

The Open AQ Platform offers information on air quality from the past 90 days. Using the location attributes from the map, we were able to query data about the air quality in that area. Google Maps API was used to get the location data of what is visible to the user and send that information to the Open AQ API.

The Google Maps provided a framelike js file, but api functionality. It helped show a map, and obtain attributes about said map.
